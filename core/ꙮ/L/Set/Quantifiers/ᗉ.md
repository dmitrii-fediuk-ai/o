# ᗉ
## Syntax
<code>ᗉ <var>S</var></code>
## Meaning
Любой / каждый элемент <var>`S`</var>.
## Parameters   
- <var>`S`</var>: is a `ꑌ⠿`.
## Rationale
Я использую такой синтаксис, чтобы удобным образом делать утверждения, которые истинны для каждого элемента <var>`S`</var>.
## Example
<code><var>A</var> ≔ ᗉ ⠿ ⟨ моё письмо клиенту ⟩</code> означает, что <var>`A`</var> — это не какое-либо конкретное моё письмо клиенту, а обозначение для любого / каждого моего письма клиентам.    
Дальше я могу удобным мне образом формулировать требования сразу ко всем моим письмам клиентам, например: 
<blockquote>
Каждый абзац <mark><var>A</var></mark> должен содержать ровно одно предложение.<br/>   
Между абзацами <mark><var>A</var></mark> не должно быть пустых строк. 
</blockquote>
Таким образом я показываю, что эти правила относятся сразу ко всем моим письмам клиентам, а не к какому-либо одному конкретному письму.  

## `ᗉ` как бинарный оператор
<code><var>A</var> ᗉ <var>S</var></code> ≡ <code><var>A</var> ≔ (ᗉ <var>S</var>)</code>
### Rationale
Используя подобный сокращённый синтаксис, я одним оператором `ᗉ` делаю сразу 2 вещи: делаю утверждение `ᗉ S` и ввожу краткое обозначение <var>`A`</var> для этого утверждения.
