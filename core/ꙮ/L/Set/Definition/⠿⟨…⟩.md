# ⠿⟨ … ⟩
## Syntax
<code>⠿⟨ <var>P</var> ⟩</code>
## Meaning
Такая запись определяет a `ꑌ⠿`  посредством <var>P</var>: формальной спецификации, которой должны соответствовать элементы этого `ꑌ⠿`. 
## Parameters   
- <var>`P`</var>: is a `ꑌℙ`.
## Example
<code>⠿ ⟨ натуральные числа ⟩</code>   
≡   
<output>множество натуральных чисел</output>
